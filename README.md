# JavaScript Programming Language

## Giới thiệu

Repository này được xây dựng nhằm phục vụ việc **học tập, thực hành và hệ thống hóa kiến thức JavaScript từ cơ bản đến
nâng cao**.

Mục tiêu của repo không chỉ là học cú pháp, mà còn:

- Hiểu **bản chất JavaScript**
- Rèn luyện **tư duy lập trình**
- Áp dụng vào **bài tập, case study và dự án thực tế**
- Làm nền tảng cho Frontend, Backend hoặc Fullstack sau này

---

## Mục tiêu học tập

Sau khi hoàn thành toàn bộ nội dung trong repository này, người học có thể:

- Nắm vững JavaScript Core (ES5, ES6+)
- Làm việc thành thạo với:
    - String, Array, Object
    - DOM, BOM, Event
    - Regular Expression
    - Storage (localStorage, sessionStorage)
    - Asynchronous JavaScript (Callback, Promise, Async/Await)
    - Fetch API và Authentication cơ bản
- Xây dựng các ứng dụng JavaScript thuần
- Sử dụng npm, module bundler (Webpack)
- Sẵn sàng học React, Vue hoặc Node.js
